JPT71 Dashboard Final Package

1. Two opening modes are supported.
   - Direct from email attachment: open `00_START_HERE.html` and review the built-in Quick Brief.
   - Full interactive package: unzip first, then open `00_START_HERE.html` or `JPT71_Voyage_Command_Center_Merged.html` from the extracted folder.

2. If `00_START_HERE.html` opens but the next dashboard does not open,
   that usually means only this one file was opened from email or ZIP preview.
   In that case, the Quick Brief still works, but the full dashboards require the ZIP to be extracted first.

3. Recommended browser: Edge or Chrome.
   Required chart scripts are embedded inside the HTML files, so charts should render without internet access.

Included files
- 00_START_HERE.html : email-safe launcher + quick brief
- JPT71_Voyage_Command_Center_Merged.html : merged main dashboard
- JPT71_KPI_Dashboard_v6.0.html : KPI source dashboard
- JPT71_Voyage_Cost_Heatmap_v4.html : cost heatmap source dashboard
- JPT71_FuelLoading_Dashboard_v6.0.html : fuel loading source dashboard
- JPT71_RootCause_Dashboard_v6.0.html : root-cause source dashboard
- JPT-reconciled_v6.0.xlsx : latest workbook
- 01_Command_Center_Preview.png : merged dashboard preview

Notes
- No standalone `.js` attachment files are included in this email-safe package.
- Required chart and data scripts are embedded directly inside the HTML dashboards.

Recommended review order
1. Open `00_START_HERE.html` first.
2. Review Quick Brief for the headline numbers.
3. If detailed review is needed, unzip the package and open the merged command center.
4. Use KPI / Heatmap / Fuel / RootCause dashboards and the workbook for trace-back.

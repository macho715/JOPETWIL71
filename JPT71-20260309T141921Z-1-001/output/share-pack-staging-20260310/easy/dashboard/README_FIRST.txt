JPT71 Dashboard + AI Package

Recommended opening method
1. Go back to the package root folder.
2. Double-click `START_JPT71_AI_DASHBOARD.bat`.
3. Wait for the browser to open automatically.

Important
- The AI panel works best when the dashboard is opened through the local starter script.
- Opening the HTML file directly can still show the dashboard, but AI calls may not work correctly.

Included files
- 00_START_HERE.html : quick launcher
- JPT71_Voyage_Command_Center_Merged.html : merged main dashboard with AI panel
- JPT71_Voyage_KPI_Dashboard_v4.html : KPI source dashboard used by the merged view
- JPT71_Voyage_Cost_Heatmap_v4.html : cost heatmap source dashboard used by the merged view
- JPT71_KPI_Dashboard_v6.0.html : extra KPI dashboard
- JPT71_FuelLoading_Dashboard_v6.0.html : extra fuel loading dashboard
- JPT71_RootCause_Dashboard_v6.0.html : extra root-cause dashboard
- JPT-reconciled_v6.0.xlsx : workbook

Recommended review order
1. Start from the package root with `START_JPT71_AI_DASHBOARD.bat`.
2. Review the merged command center first.
3. Use the AI panel for summaries, ranking, comparison, and action ideas.
4. Use KPI / Heatmap / Fuel / RootCause dashboards and the workbook for trace-back.
